Selector querySelector()



En w3schools aparecen muy buenos ejemplos.

Esta propiedad se encarga de obtener el primer elemento del documento con la clase indicada.

El método querySelector() devuelve el primer elemento que coincide con un selector CSS especificado en el documento.

Nota: El método querySelector() sólo devuelve el primer elemento que coincide con los selectores especificados. Para devolver todas las coincidencias, utilice el método querySelectorAll ().

Si el selector coincide con un ID en el documento que se utiliza varias veces (Observe que un "id" debe ser único dentro de una página y no debe utilizarse más de una vez), devuelve el primer elemento coincidente.

Ejemplo de la propiedad querySelector()

document.querySelector(.estiloTexto)

De esa forma indicamos que la propiedad <h1> titulo va a contener la clase .estiloTexto y al ser el primero mediante querySelector se le añade el estilo.

Nota Extra: Si el primer elemento no contiene la clase añadida en "querySelector()" se encarga de buscar la siguiente, si esta ubicada de ultimo se añade la clase a ese elemento, si hay dos elementos con esa clase solo se añade al elemento que este de primero y si no encuentra la clase da como resultado null.
