querySelectorAll()

Provando JavaScript

En w3schools hay buenos ejemplos

El método querySelectorAll() devuelve todos los elementos del documento que coinciden con un selector CSS especificado, como un objeto NodeList estático.

El objeto NodeList representa una colección de nodos. Se puede acceder a los nodos por números de índice. El índice empieza en 0.

Sugerencia: Puede utilizar la propiedad length del objeto NodeList para determinar el número de elementos que coincide con el selector especificado, luego puede recorrer todos los elementos y extraer la información que desea.

Este metodo especifica uno o más selectores CSS para que coincidan con el elemento. Éstos se utilizan para seleccionar elementos HTML basados en su id, clases, tipos, atributos, valores de atributos, etc.

Nota: Para selectores múltiples, separe cada selector con una coma.

Ejemplo:

    // Get all <p> elements in the document
    var x = document.querySelectorAll("p");

    // Set the background color of the first <p> element
    x[0].style.backgroundColor = "red";

Como seleccionar todos los elementos con este metodo

      var x = document.querySelectorAll(".example");
      var i;
      for (i = 0; i < x.length; i++) {
      x[i].style.backgroundColor = "red";
      }
